    串口通信

    串口通信用于链接电脑与单片机的通信方式，通过串口通信可以实现电脑与单片机之间的数据传输。通过串口通信与电子单片机传递角度信息，实现对电机的控制。

    在 armTool 中由armSerial类实现串口通信。在其中定义了有关串口链接，串口发送等函数。

    armSerial类

    构造方法

    方法原型：__init__(self, port: str = None)

    串口初始化函数，用于初始化串口。在初始化串口时，可传入指定串口号，串口号为字符串类型。若未传入串口号则默认为空，并自动获取可使用串口名称。常见串口号在 Windows 下为COM1、COM2等，Linux 下为/dev/ttyUSB0等。

    if port:
    self.port = port
else:
    self.port = self.get_port()

    其中self.get_port()函数用于获取当前可用串口号。

    当串口号获取成功后，尝试打开串口，为发送信息做准备。如果成功打开串口则会在终端打印[INFO] Serial(XXX) is open!，否则打印[WARN] Serial (XXX) is closed!。（XXX 为串口号）

    if self.ser.isOpen():
    print(
        "\033[0;32m[INFO] Serial(\033[3;32m{}\033[0;32m) is open! \033[0;0m".format(
            self.port
        )
    )
else:
    print(
        "\033[0;33m[WARN]Serial \033[4;31m({})\033[0;31m is closed!\033[0;0m".format(
            self.port
        )
    )

    串口名称获取方法

    方法原型：get_port(self)

    通过调用find_available_serial_ports()获取当前所有可用串口名称，并返回第一个可用串口名称。

    如果未找到可用串口则抛出异常[ERRO] No serial port found!。

    try:
    port = self.find_available_serial_ports()
    if port == []:
        raise Exception("\033[0;31m[ERRO] No serial port found!\033[0;0m")
    return port[0]
except Exception as e:
    exit(e)

    静态方法获取串口名称

    方法原型：find_available_serial_ports()

    通过遍历串口号，获取当前所有可用串口名称，并返回一个列表。

    在 Windows 下，串口号为COM1、COM2等，Linux 下为/dev/ttyUSB0等。

    @staticmethod
def find_available_serial_ports():
    if os.name == "posix":
        ports = glob.glob("/dev/tty[A-Za-z]*")
    elif os.name == "nt":
        ports = ["COM%s" % (i + 1) for i in range(256)]
    else:
        raise OSError("Unsupported platform")
    available_ports = []
    for port in ports:
        try:
            ser = serial.Serial(port)
            ser.close()
            available_ports.append(port)
        except (OSError, serial.SerialException):
            pass
    return available_ports

    数据发送方法

    方法原型：send_data(self, data: bytes)

    用于向串口发送数据，传入参数为字节类型，并在控制台以十六进制的格式打印发送的数据，开头为[INFO] Data sent:。如果再发送过程中，出现错误则会抛出异常[WARN] Data not sent。

    try:
    self.ser.write(data)
    print("\033[0;32m[INFO] Data sent: \033[0;0m", end="")
    for i in range(len(data)):
        if i == 3:
            print("\033[0;35m", end="")
        if i == len(data) - 2:
            print("\033[0;36m", end="")
        if 3 < i < len(data) - 3:
            print("\033[0;33m", end="")
        if i == len(data) - 3:
            print("\033[0;3{}m".format(data[i] - 70 + 1), end="")
        print("{:02X}\033[0;0m".format((data[i])), end=" ")
    print("\n", end="")
except:
    raise Exception("\033[0;31m[WARN] Data not sent\033[0;0m")

    串口信息

    在比赛时通常用于控制机械臂运动的串口号为/dev/ttyUSB2，波特率为115200。

    通信时使用的数据帧见：数据帧格式

    注意事项

    1. 串口通信时，需保证串口号以及波特率的正确性，否则会导致通信失败。
    2. 如果存在多个可用串口，可通过拔插数据线缆的方式，查看串口列表的变化，以确定机械臂所使用的串口。
    3. 在Linux系统下，若串口无法打开，可通过sudo chmod 777 /dev/ttyUSB*命令修改串口权限。
    4. 在Windows系统下调试串口，需要提前安装相关驱动程序，否则无法正常使用串口通信。
    5. 在Windows系统调试时，注意在设备管理器中查看串口号，以确定串口号的正确性。
